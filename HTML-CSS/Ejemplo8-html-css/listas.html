Glosario de velocidades

Definicion Kilometros por hora:
El kilómetro por hora es una unidad de medida de velocidad, tanto escalar como vectorial. Un objeto viaja a una velocidad media de 1 km/h si se desplaza 1 kilómetro en una hora El símbolo utilizado para esta unidad es km/h

Definicion millas por hora:
La milla por hora es una unidad de medida de velocidad que expresa el número de millas internacionales recorridas por hora
